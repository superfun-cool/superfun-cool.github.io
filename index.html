Web Scraping? What's that?
Put simply, web scraping is when you use software to automatically read data from a website. It sounds boring, but it's actually super useful! Its most common use case is for tracking price changes on websites such as Amazon or eBay, but there's a bunch more you can do with it. You'll be exploring several different examples and use cases for web scraping!
